## ğŸ“š **API ENDPOINTS**

### **Exam Management API:**

**Exams:**
- `GET /api/exams.php?resource=exams` - List all exams
- `GET /api/exams.php?resource=exams&id={id}` - Get exam details
- `POST /api/exams.php?resource=exams` - Create exam
- `PUT /api/exams.php?resource=exams&id={id}` - Update exam
- `DELETE /api/exams.php?resource=exams&id={id}` - Delete exam

**Schedules:**
- `GET /api/exams.php?resource=schedules&action=by_exam&exam_id={id}` - Get exam schedules
- `GET /api/exams.php?resource=schedules&action=by_date&date={date}` - Get schedules by date
- `POST /api/exams.php?resource=schedules` - Create schedule
- `PUT /api/exams.php?resource=schedules&id={id}` - Update schedule

**Seating:**
- `GET /api/exams.php?resource=seating&action=by_schedule&schedule_id={id}` - Get seating
- `GET /api/exams.php?resource=seating&action=generate&schedule_id={id}` - Auto-generate seating
- `POST /api/exams.php?resource=seating` - Assign seat
- `PUT /api/exams.php?resource=seating&id={id}&action=mark_attendance` - Mark attendance

**Results:**
- `GET /api/exams.php?resource=results&action=by_schedule&schedule_id={id}` - Get results
- `GET /api/exams.php?resource=results&action=by_student&student_id={id}&exam_id={id}` - Student results
- `POST /api/exams.php?resource=results` - Enter marks
- `PUT /api/exams.php?resource=results&action=publish&schedule_id={id}` - Publish results

**Statistics:**
- `GET /api/exams.php?resource=statistics&schedule_id={id}` - Get exam statistics
