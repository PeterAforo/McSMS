## ğŸ“š **API ENDPOINTS**

### **Transport API:**

**Vehicles:**
- `GET /api/transport.php?resource=vehicles` - List vehicles
- `GET /api/transport.php?resource=vehicles&id={id}` - Vehicle details
- `POST /api/transport.php?resource=vehicles` - Add vehicle
- `PUT /api/transport.php?resource=vehicles&id={id}` - Update vehicle

**Drivers:**
- `GET /api/transport.php?resource=drivers` - List drivers
- `POST /api/transport.php?resource=drivers` - Add driver
- `PUT /api/transport.php?resource=drivers&id={id}` - Update driver

**Routes:**
- `GET /api/transport.php?resource=routes` - List routes
- `GET /api/transport.php?resource=routes&id={id}` - Route with stops
- `POST /api/transport.php?resource=routes` - Create route

**Assignments:**
- `GET /api/transport.php?resource=assignments&action=today` - Today's assignments
- `POST /api/transport.php?resource=assignments` - Create assignment
- `PUT /api/transport.php?resource=assignments&id={id}&action=update_status` - Update status

**GPS Tracking:**
- `GET /api/transport.php?resource=tracking&action=live` - Live positions
- `GET /api/transport.php?resource=tracking&action=history&vehicle_id={id}&date={date}` - History
- `POST /api/transport.php?resource=tracking` - Update location

**Student Transport:**
- `GET /api/transport.php?resource=student_transport&action=by_route&route_id={id}` - Students on route
- `GET /api/transport.php?resource=student_transport&action=by_student&student_id={id}` - Student transport
- `POST /api/transport.php?resource=student_transport` - Register student

**Maintenance:**
- `GET /api/transport.php?resource=maintenance&action=upcoming` - Upcoming maintenance
- `GET /api/transport.php?resource=maintenance&action=by_vehicle&vehicle_id={id}` - Vehicle history
- `POST /api/transport.php?resource=maintenance` - Schedule maintenance
